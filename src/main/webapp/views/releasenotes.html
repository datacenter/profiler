01 Overview

      Network Sizer is a Web based application which helps in sizing the Network infrastructure of a datacenter for ACI or Nexus 9000 based deployments.

02 How to use Network Sizer

      Network Sizer is hosted at https://t459jj4xprd-nwsserver.cloudapps.cisco.com/networksizer, https://nwsizer.cloudapps.cisco.com/networksizer
  
      Open your Internet browser such as Chrome or Firefox and visit the above URL.
      Enter your Cisco user credentials.
      First page displays a list of all projects previously created by the user.
      Start a new project either by creating a New project by clicking "+" button or clone (copy) an existing project.
      Once you create a project, detailed view of the project comes up.
      Select Physical or Logical Sizer details to add elements to project.
   
      For additional details, refer user guide by clicking drop down menu on top right corner of the page.

03 Known Issues

      1. Application is tested on Chrome and Firefox browsers.
   
      Workaround:Since the application uses standard HTML/CSS components, it is expected to work on
                          other browsers/operating systems.

04 How to log defects

      Select "Feedback" link by clicking dropdown menu on the top right corner of the page. Fill the form and submit.
